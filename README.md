## 📖 Project Overview
This project was developed during my internship at EISystems Services. The objective was to build a robust predictive model for house prices using **Multiple Linear Regression**.

### Features Used:
- **Area** (in square feet)
- **Bedrooms** (number of bedrooms)
- **Age** (age of the house in years)

### Target Variable:
- **Price** (market value of the house)

## 🧠 Algorithm & Tools
| Category | Technology |
|----------|------------|
| Algorithm | Multiple Linear Regression |
| Language | Python 3.x |
| Libraries | Pandas, Scikit-learn |
| Environment | Jupyter Notebook |

## 🚀 How to Run This Project

### Prerequisites
```bash
pip install pandas scikit-learn
```
### Steps to Execute
1. Make sure Multi.csv is in the same directory
2. Run the Python script:

```bash
 python house_price.py
```
### 📈 Model Results
 The model will output:

Coefficients for each feature (area, bedrooms, age)

Intercept value

Predicted price for a new house
