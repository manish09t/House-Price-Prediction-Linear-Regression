## 📖 Project Overview
This project was developed during my internship at EISystems Services. The objective was to build a robust predictive model for house prices using **Multiple Linear Regression**.

### Features Used:
- **Area** (in square feet)
- **Bedrooms** (number of bedrooms)
- **Age** (age of the house in years)

### Target Variable:
- **Price** (market value of the house)
