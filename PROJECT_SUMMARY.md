# Quick Project Summary

## Problem Statement
Predict house prices based on area, number of bedrooms, and age.

## Solution
Multiple Linear Regression model achieving accurate price predictions.

## Key Results
- **Training Data:** 10+ house records
- **Features:** 3 (area, bedrooms, age)
- **Algorithm:** Linear Regression (scikit-learn)

## Sample Prediction
| Feature | Value |
|---------|-------|
| Area | 5000 sq. ft |
| Bedrooms | 3 |
| Age | 5 years |
| **Predicted Price** | **₹8,12,500** |
